# Rahul AI: The Unleashed Core

Rahul AI is an advanced, high-performance AI assistant designed for autonomy, efficiency, and roleplay. Created by **Rahul Sir**, this assistant is built for users who demand a raw, powerful, and modern AI experience without the restrictions of standard assistants.

## 🚀 Key Features

*   **Premium Claude.ai-Style GUI:** Modern, dark-glassmorphism chat interface with smooth animations.
*   **Realistic Voice Interaction:** Powered by `edge-tts` (Indian Male `hi-IN-MadhurNeural`), providing natural and lag-free speech.
*   **Advanced AI Brain:** Modular engine supporting multiple LLM backends (G4F, Ollama) for powerful NLP.
*   **Modular Skill System:** Easily extensible system for Weather, System Control, Notes, and more.
*   **God Mode (YOLO):** Minimal friction and auto-managed permissions for a seamless experience.
*   **Cross-Platform:** Runs natively on Windows and optimized for Termux (Android).

## 🛠️ Installation & Setup

### For Windows:
1. Ensure you have Python installed.
2. Run `setup.bat` to install all dependencies.
3. Launch the assistant using `start_rahulai.bat`.
4. Open `http://localhost:8000` in your browser.

### For Termux (Android):
```bash
# 1. Update Termux
pkg update && pkg upgrade -y
# 2. Install Python & dependencies
pkg install python git -y
# 3. Clone the repo
git clone https://github.com/devrahulmaida-sketch/Rahul-AI-Core.git
cd Rahul-AI
# 4. Install dependencies
pip install -r backend/requirements.txt
# 5. Launch
python backend/main.py
```
